# 成本服务器项目 (Cost Server)

这是一个用于管理和计算项目成本的Web应用。

## 功能

- 用户认证和授权
- 项目数据管理
- 成本计算和分析
- 报表生成

## 技术栈

- Python
- Flask
- SQLite/MySQL
- HTML/CSS/JavaScript

## 安装

1. 克隆仓库
```
git clone https://github.com/wagyog2023/cost_server66.git
cd cost_server66
```

2. 安装依赖
```
pip install -r requirements.txt
```

3. 运行应用
```
python app.py
```
